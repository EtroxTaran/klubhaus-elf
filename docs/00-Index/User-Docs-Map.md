# User Docs Map

User docs are written for players. They explain shipped behavior from the
player's point of view.

## Current Hub

- [User Docs README](../70-User-Docs/README.md)

## Future Structure

- Getting started and PWA install.
- How to play.
- Mechanics reference.
- FAQ.
- Player-facing changelog.
- Accessibility.

## Rule

Do not infer implementation behavior from user docs. Generate or update user
docs from accepted ADRs, approved game design notes, approved feature specs, and
shipped code.
