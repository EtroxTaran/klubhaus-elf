# Glossary

- **Beat**: One issue completed through plan, red, green, refactor, docs, PR, CI, and deployment.
- **Career save**: Pre-reopen term for one manager career save; persistence
  details remain draft until re-ratified.
- **Vault**: The `docs/` directory used as an Obsidian knowledge base.
- **IP-clean data**: Generated names/assets that do not copy real club, player, or league IP.
- **Intent layer**: `draft`/`idea` notes — recognized current best thinking. Read for direction and to avoid re-deciding; never implemented from or cited as settled. Distinct from `superseded`/`archived` (history only).
- **Reopened decision**: An ADR/GDDR/system note reset to `draft` on
  2026-05-27; historical accepted/approved wording is review context only.
- **Aurelia Premier**: Pre-reopen in-world league/design-system working title.
- **Sonntagszeitung (Direction A)**: Pre-reopen visual direction memory: cream
  paper, dark ink, single scarlet accent, Newsreader/Inter/JetBrains Mono.
- **Design handoff bundle**: A Claude Design export (gzip-tar) of the prototype,
  snapshotted under `design/handoff/<date>/`.
- **design-sync**: The `pnpm sync:design` workflow that snapshots and diffs
  design exports; see [[../30-Implementation/design-sync-workflow]].
