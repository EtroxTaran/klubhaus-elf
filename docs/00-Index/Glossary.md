# Glossary

- **Beat**: One issue completed through plan, red, green, refactor, docs, PR, CI, and deployment.
- **Career save**: A local IndexedDB-backed game state for one manager career.
- **Vault**: The `docs/` directory used as an Obsidian knowledge base.
- **IP-clean data**: Generated names/assets that do not copy real club, player, or league IP.
- **Aurelia Premier**: The IP-clean in-world league and working title of the game's design system.
- **Sonntagszeitung (Direction A)**: The chosen visual direction — cream paper, dark ink, single scarlet accent, Newsreader/Inter/JetBrains Mono.
- **Design handoff bundle**: A Claude Design export (gzip-tar) of the prototype, snapshotted under `design/handoff/<date>/`.
- **design-sync**: The `pnpm sync:design` workflow that snapshots and diffs design exports; see [[../30-Implementation/design-sync-workflow]].
