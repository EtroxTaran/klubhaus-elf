# Session Handoffs

This folder is short-term working memory for agents. Use it for daily or
session-based context that helps the next agent resume safely.

Session notes are not authoritative. Promote durable decisions into accepted
ADRs, approved game design or feature notes, current-state pages, or
implementation docs.

## Naming

Use `YYYY-MM-DD-topic.md`, for example:

- `2026-05-16-vault-memory-system.md`
- `2026-05-17-match-engine-planning.md`

## Required Sections

- Goals
- Completed
- Open Tasks
- Decisions Made
- Blockers
- Durable Notes Updated
- Promotion Needed
