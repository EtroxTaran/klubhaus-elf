# Introduction

`soccer-manager` is an offline-ready football management PWA focused on a
Create-a-Club Roguelite MVP, deterministic simulation, and IP-clean generated
game data. The MVP is hybrid-online; selective offline-first singleplayer is a
planned future capability.
