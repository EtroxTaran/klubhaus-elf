# ADR-0001: Tech Stack

## Decision

Use TanStack Start, React, Tailwind/shadcn, SurrealDB, Dexie, pnpm, Biome,
Vitest, Playwright, Docker, Dokploy, and Cursor Cloud Agent workflows.

## Consequences

The stack is optimized for offline-first PWA gameplay and agent-friendly CI.
TanStack Start beta volatility remains an active risk.
