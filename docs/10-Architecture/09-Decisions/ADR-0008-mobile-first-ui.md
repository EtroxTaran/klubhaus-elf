# ADR-0008: Mobile-first UI Approach

## Decision

Design primary gameplay flows for mobile first while preserving desktop-friendly
responsive layouts.

## Consequences

Navigation, tables, simulation controls, and save flows must remain usable on
small touch screens and with keyboard navigation.
