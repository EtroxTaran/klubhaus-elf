# Raw - DriftConsumerPolicyRef real-world football governance research

## Prompt

Research real-world football precedents that should inform a fictional football
manager game event schema for long-save world drift: forced owner sale,
ownership/fit-and-proper gates, sanctions/transfer restrictions, financial
sustainability/licensing rules, administration/insolvency, and youth/talent
diffusion. Focus on what metadata a game event should carry for consumers, not
exact real-law implementation.

## Perplexity Capture

Perplexity returned useful metadata families, but several citations were
cross-sport, secondary, or weak. They are preserved here as discovery material,
not as canonical football-law authority.

Raw schema implications returned:

- Forced-sale events should expose trigger category, initiating body, procedure
  type, status, response/appeal windows, operational restrictions during the
  process, sale outcome, reputation/fan/board consequences and linked
  investigation ids.
- Ownership-gate events should expose applicant owner, transaction type, checked
  disqualifying factors, screening depth, result, failed factors, imposed
  conditions and follow-up forced-sale links.
- Sanction and transfer-restriction events should expose allegation categories,
  sanction type, basis, fine/points/embargo/registration/spending constraints,
  start/end/review dates and conditions for lifting.
- Financial-sustainability/licensing events should expose licensing body,
  license scope, reporting window, aggregate financial inputs, compliance status,
  breached categories, license decision, conditions and sanction links.
- Administration/insolvency events should expose distress type, debt/revenue
  ratio, wage/tax arrears, proceeding type, administrator/creditor categories,
  points deduction, registration restrictions and exit outcome.
- Youth/talent diffusion should be a follow-up hint: academy investment changes,
  forced sale waves, youth/homegrown quota shocks and regional talent pipeline
  changes should link causally to drift, sanctions and ownership events.
