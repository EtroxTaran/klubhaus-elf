# db/ — SUPERSEDED (2026-05-19, ADR-0021)

The SurrealDB schema and `.surql` migrations in this directory are **superseded**
by [ADR-0021](../docs/10-Architecture/09-Decisions/ADR-0021-revised-tech-stack.md).

PostgreSQL + Drizzle is the system of record. The authoritative schema is now
schema-as-TypeScript in **`packages/db/src/schema.ts`**; forward-only migrations
are generated by drizzle-kit into `packages/db/migrations/` (via
`packages/db/drizzle.config.ts`). The Zod mirror is `packages/db-schema`.

The full ADR-0004 domain model rework from the SurrealDB model to Drizzle is the
next engineering wave (the repo operates doc-ahead-of-code). Files here are kept
for history only — do not apply them.

SurrealDB may return only as an optional **additive** realtime/graph engine
behind the [ADR-0023](../docs/10-Architecture/09-Decisions/ADR-0023-realtime-transport.md)
`RealtimeTransport` interface — never as the system of record.
